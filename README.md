# PythonでOCR

## 概要

このリポジトリは、PythonでOCRを行うためのサンプルコードを提供します。

## 環境

- Python 3.12
- Tesseract 5.5.0
- OpenCV 4.10.0

## インストール

```bash
pip install -r requirements.txt
```

## 使い方

```bash
python main.py
```

実行すると、`test.png`を読み込んで、OCRを行います。  
OCRは、英語のみ対応しています。
